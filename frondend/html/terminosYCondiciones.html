TERMINOS Y CODICIONES

Aceptación de los Términos y Condiciones

1. Al acceder a este sitio web, el usuario acepta los presentes Términos y Condiciones de Uso.
2. Para poder realizar reservas, el usuario debe tener una cuenta de usuario en el sitio web.
3. El usuario permitira que el sitio web solicite y utilice la informacion solictada para poder realizar la reserva.
4. El usuario acepta que el sitio web pueda enviarle correos electronicos con informacion sobre el estado de su reserva.
5. El usuario debera tener registrada una tarjeta de credito para poder realizar la reserva.
6. El usuario acepta que el sitio web pueda cobrarle el monto de la reserva a la tarjeta de credito registrada.
7. El usuario no podra modificar reservas con menos de 5 dias de anticipacion a la fecha de entrada.
8. En caso de modificar o cancelar la reserva, el usuario recibira un cupon del 15% de descuento para su proxima reserva( no expirable).
9. En ningun caso se realizara un reembolso del monto de la reserva.
10. El usuario puede registrar uno o varios negocios en el sitio web.
11. El administrador del sitio web se reserva el derecho de aceptar, rechazar o eliminar negocios que no cumplan con los estandares de calidad del sitio web.
